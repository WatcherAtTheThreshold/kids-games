Parent Information

About Kids Games

Educational games designed for toddlers ages 3-5. Each game focuses on basic learning skills:

  • Color Pop: Color recognition and listening skills
  • Animal Peekaboo: Animal identification and sounds
  • Bug Count: Early counting and number recognition

Game Features

  • ✅ No fail states - gentle encouragement only
  • ✅ 60-180 second sessions perfect for attention spans
  • ✅ Large touch targets for small fingers
  • ✅ Audio can be toggled on/off
  • ✅ Progress saved automatically

Privacy & Safety

All games run locally - no data collection, no ads, no external connections required.
